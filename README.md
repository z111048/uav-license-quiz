# 無人機學科線上練習系統

這是一個簡易的網頁版互動式練習題庫，專為準備無人機學科考試設計。

## 功能特色

- **練習設定**：
  - 可自選特定章節進行練習。
  - 支援設定練習題數（5、10、20、50 題或全部）。
  - 可開啟「作答後立即顯示正解」功能。

- **作答模式**：
  - **計時測驗**：每題有倒數計時，模擬考試壓力。
  - **閱讀模式**：直接瀏覽題目與答案，適合快速複習。
  - **錯題回顧**：練習結束後可針對答錯的題目進行檢討。

- **技術架構**：
  - 純 HTML/CSS/JavaScript 實作，無需後端伺服器。
  - 使用 Tailwind CSS 進行切版。
  - 題庫資料儲存於 `question_bank.json`，方便維護與擴充。

## 如何使用

1. 請確保資料夾中包含 `index.html` 與 `question_bank.json` 檔案。
2. 直接點擊兩下 `index.html` 以瀏覽器開啟即可開始使用。
3. 若需更新題庫，請依 JSON 格式編輯 `question_bank.json` 檔案。

## 題庫格式說明

題庫檔案 `question_bank.json` 結構如下：

```json
[
  {
    "id": 1,
    "question": "題目敘述...",
    "options": {
      "A": "選項 A",
      "B": "選項 B",
      "C": "選項 C",
      "D": "選項 D"
    },
    "answer": "正確選項 (如 A)",
    "chapter": "章節名稱"
  }
]
```
